## How to Add These to Notion

1. **Open your Notion database**: "Mileshope.com"

2. **For each post idea**, create a new page with:
   - **Title**: The post title from above
   - **Status**: "Draft" (you'll change to "Published" when ready)
   - **Content**: Write your content (or start with an outline)
   - **Publish Date**: Choose when you want it to go live
   - **Tags**: Relevant tags (e.g., "bazi", "tarot", "beginner-friendly")
   - **Category**: Main category (e.g., "Bazi", "Tarot", "Personal Growth")
   - **SEO Description**: Brief summary for search engines

3. **Write the post** in the Content field

4. **When ready**, change Status to "Published"

5. **Run sync**:
   ```bash
   set -a && source .env && set +a && python3 sync.py
   ```

6. **Preview**: Refresh http://127.0.0.1:1111/blog/

## Content Strategy Tips

### Start With These:
- 1 Bazi beginner post
- 1 Tarot beginner post
- 1 Personal story/perspective post

### Publishing Schedule:
- **Weekly**: 1-2 posts per week is sustainable
- **Consistency**: Pick a day and stick to it
- **Quality over quantity**: Better to publish less, but well-written

### Content Mix:
- 40% Educational (How-to, guides)
- 30% Inspirational (Philosophy, stories)
- 20% Practical (Spreads, techniques)
- 10% Personal (Your journey, reflections)

### SEO Keywords to Include:
- "Bazi chart", "Four Pillars astrology"
- "Tarot reading", "Tarot card meanings"
- "Chinese astrology"
- "Personal growth", "spiritual development"
- "Divination", "self-awareness"
